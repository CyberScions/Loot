    GrabME - Extract Sensitive information from a file.

    Usage: ./grabme.py [FILE]

    What can it extract ?:

    Links
    hash values
    email addresses
    ipv4, ipv6 addresses
    bitcoin wallet addresses
    MAC addresses with : or - (deliminators)
    USA Based Telephone, Social Security and Major Credit Card numbers.
